Description

Your own app store

Instructions

1. config param apps_dir in odoo config file (odoo.conf) 2. Create a folder named "apps_dir" like param apps_dir in the config file e.g. "/mnt/extra-addons/apps_dir"
3. Upload your module folders to the newly created 'apps_dir' folder
4. Go to Settings->App Store->Update App Store and click 'Update Module List'
5. View your modules on the App Store menu on your website
